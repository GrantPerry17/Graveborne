# Graveborne

## Overview
Graveborne is a Unity multiplayer arena PvP game (Free-for-All). Built in Unity 5000.2.0b13 using Photon PUN for networking.

## How to open
1. Clone the repo.
2. Open in Unity Hub (Unity 2022.x recommended).
3. Let Unity regenerate Library folder.

## Notes
- Do not commit Library/, Temp/, Build/ folders.
